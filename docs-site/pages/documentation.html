Overview

Yapzone is a multi-part project designed to bring real-time communication features to your browser with ease. It consists of:

  • 🧩 A Chrome extension for seamless chat and audio calling.
  • 🖥️ A Node.js backend using Socket.IO to handle real-time communication.
  • 📚 A static documentation site explaining how the extension works and how to contribute.

Created with ❤️ by the homies, Yapzone is designed to make collaboration fun and easy!

Features

  • 💬 Real-time messaging using WebSockets (Socket.IO)
  • 🎧 Audio call functionality for live communication
  • 🧩 Chrome extension UI for easy interaction
  • 📚 Built-in static documentation website

Folder Structure

Here is the structure of the Yapzone project:

yapzone/
├── 📄 docs-site     # Static documentation (HTML, CSS, JS)
├── 🧩 extension     # Chrome extension source (manifest, popup, scripts)
├── 🖥️ server        # Backend (Node.js + Socket.IO)
└── 📘 readme.md     # Project README

docs-site/: Contains the documentation files built with HTML, CSS, and JS.

extension/: Includes the source code for the Chrome extension (manifest, popup UI, scripts).

server/: The backend code (Node.js + Socket.IO) that handles the real-time chat and audio call features.

readme.md/: The project's main README file that includes an overview and instructions.

Getting Started

To get started with Yapzone, follow the instructions below:

🔹 Load the Chrome Extension

1. Open Chrome and go to chrome://extensions/

2. Enable Developer Mode at the top-right of the Extensions page.

3. Click Load unpacked and select the extension/ folder from the Yapzone project.

4. The extension will now be added to your browser and is ready for use!

🔸 Run the Backend Server

To run the backend server and enable real-time messaging and calling:

cd server
npm install
node index.js

This will start the Node.js server, enabling the chat and call features in the extension. Make sure the backend is running before using the extension.

Tech Stack

  • 💻 **Frontend**: HTML, CSS, JavaScript
  • ⚙️ **Backend**: Node.js, Socket.IO for real-time communication
  • 🔌 **Chrome Extension**: Using Chrome APIs (popup, background scripts, content scripts)
  • 📝 **Documentation**: Static HTML, CSS, and JavaScript for documentation site

  4. 🖋️ Make your changes and commit them: git commit -m "Add feature-name"
  5. 🚀 Push to your branch: git push origin feature-name
  6. 🔁 Open a Pull Request

We encourage open-source contributions to make this project even better!
